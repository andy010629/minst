## Introduction
A web canvas that you can draw and see the MNIST classification result distribution.

## Requirements
* Python 3.6+
* Pytorch 0.4.0+
* Pandas 0.23.4+
* NumPy 1.14.3+
## Usage
  Train the model:
      python3 main.py
  Convert pth to onnx:
      python3 convert.py
